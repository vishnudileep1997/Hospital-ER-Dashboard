# Hospital Emergency Room Dashboard

## Project Overview
This Excel dashboard analyzes hospital emergency room data and provides insights into patient trends, wait times, satisfaction scores, admission status, gender distribution, and department referrals.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Dashboard Features
- KPI Cards
- Monthly Filters
- Age Group Analysis
- Patient Satisfaction Metrics
- Department Referral Analysis
